## 2. 本次开发任务

### 任务: Runtime 事件系统 C 实现

**选择理由**:
1. 昨天 Codegen 已完成事件系统代码生成（Publish/Subscribe/SerialProcess）
2. Runtime 函数仅有外部声明，缺少实际实现
3. Runtime 事件系统是编译器可运行演示的关键依赖

### 实现内容

**新增文件**: `src/runtime/claw_event.h` (~209 行)

#### 1. 事件系统核心 (claw_event_t)
- 支持 256 个事件槽位
- 每个事件最多 32 个订阅者
- 全局事件系统状态管理

#### 2. 事件 API 函数
- `claw_event_get_or_create()` - 查找或创建事件
- `claw_event_subscribe()` - 订阅事件，返回 subscriber_id
- `claw_event_dispatch()` - 分发事件，调用所有 handler
- `claw_event_unsubscribe()` - 取消订阅
- `claw_event_print_status()` - 打印系统状态

#### 3. 串行进程 API (claw_process_t)
- 支持 64 个进程
- `claw_process_init()` - 注册进程
- `claw_process_run()` - 运行指定进程

#### 4. 运行时初始化
- `claw_runtime_init()` - 初始化运行时
- 自动初始化事件系统
