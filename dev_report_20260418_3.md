## 技术细节

### 编译流程
```
AST Node → BytecodeCompiler::compileNode()
    ↓
Statement → 各种 compileXxx() 方法
    ↓
Expression → compileExpression() → 指令序列
    ↓
Function/Module → bytecode::Module
```

### 指令发射
- 使用统一的 `emit()` 方法
- 支持常量池索引优化
- 支持局部变量快速加载 (LOAD_LOCAL_0, LOAD_LOCAL_1)

### 跳转管理
- 前向跳转: emit_jump() → patch_jump()
- 循环支持: LoopContext 管理 break/continue

### 操作符映射
- 算术运算: Op_plus → IADD, Op_star → IMUL, etc.
- 比较运算: Op_eq → IEQ, Op_lt → ILT, etc.
- 一元运算: Op_minus → INEG, Op_bang → NOT, etc.
