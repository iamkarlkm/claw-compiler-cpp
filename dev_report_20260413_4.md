## 2. 本次开发任务

### 任务: Interpreter For 循环数组/张量/字符串迭代支持

**选择理由**:
1. Interpreter 中 For 循环仅支持 Range 表达式 (1..10)
2. 缺少对数组、张量、字符串的迭代支持
3. 这是用户高频使用场景，优先级高

### 实现内容

**修改文件**: `src/interpreter/interpreter.h`

**新增功能** (~105 行):
1. **数组/张量迭代**: 支持 RuntimeValue 类型的数组迭代，自动提取元素
2. **Vector 迭代**: 支持 `std::vector<Value>` 类型迭代
3. **字符串迭代**: 支持逐字符迭代 (char by char)
4. **1-based 索引语义**: 保持 Claw 语言 1-based 索引特性

**关键代码**:
```cpp
// 支持数组/张量迭代
if (std::holds_alternative<RuntimeValue>(arr_val)) {
    auto arr = std::get<RuntimeValue>(arr_val);
    for (size_t idx = 0; idx < arr.array.size(); idx++) {
        RuntimeValue loop_var = arr.array[idx];
        loop_var.scalar = static_cast<int64_t>(idx + 1);  // 1-based
        runtime.set_var(var_name, loop_var);
        execute_block(for_stmt->get_body());
    }
}

// 支持字符串迭代 (逐字符)
if (std::holds_alternative<std::string>(arr_val)) {
    auto str = std::get<std::string>(arr_val);
    for (size_t idx = 0; idx < str.size(); idx++) {
        RuntimeValue loop_var;
        loop_var.type_name = "char";
        loop_var.scalar = static_cast<int64_t>(str[idx]);
        runtime.set_var(var_name, loop_var);
        execute_block(for_stmt->get_body());
    }
}
```
