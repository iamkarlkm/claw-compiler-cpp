## 本次开发任务

### 功能: REPL (交互式解释器)

**选择理由**: 
- REPL 是编译器生态系统的核心工具，提供交互式编程体验
- 对编译器开发和测试至关重要
- 是一个完整的、独立的功能模块 (>=500 行代码)

**设计目标**:
1. 支持 Claw 语言语法的交互式执行
2. 提供丰富的 REPL 命令
3. 支持多行输入和代码补全
4. 集成词法分析、语法分析、类型检查和解释执行

### 实现文件

1. **src/repl/repl.h** (~230 行)
   - InputBuffer: 输入缓冲区管理（支持多行输入）
   - CompletionEngine: 代码补全引擎
   - ResultFormatter: 结果格式化器
   - REPL 类: 主 REPL 实现

2. **src/repl/repl.cpp** (~650 行)
   - 完整的 REPL 实现
   - 14 个 REPL 命令
   - readline 集成（历史记录、行编辑）
   - 与现有编译器组件集成

3. **src/repl_main.cpp** (~10 行)
   - 独立 REPL 入口点

4. **src/repl_simple.cpp** (~350 行)
   - 简化版 REPL（无 readline 依赖）
   - 用于测试和演示

### REPL 命令列表

| 命令 | 描述 |
|------|------|
| :help, :h | 显示帮助信息 |
| :quit, :q | 退出 REPL |
| :clear, :c | 清屏 |
| :history, :hist | 显示输入历史 |
| :info <name> | 显示变量/函数信息 |
| :type <expr> | 显示表达式类型 |
| :load <file> | 加载并执行文件 |
| :save <file> | 保存历史到文件 |
| :reset, :r | 重置环境 |
| :debug, :d | 切换调试模式 |
| :ast <expr> | 显示 AST |
| :tokens <expr> | 显示词法分析结果 |
| :time <expr> | 测量执行时间 |
| :version, :v | 显示版本信息 |
| :complete <str> | 代码补全建议 |

## 技术实现

### 核心架构
```
+-------------+     +-------------+     +-------------+
|  Input      | --> |   Lexer     | --> |   Parser    |
|  Buffer     |     |  (词法分析)  |     |  (语法分析) |
+-------------+     +-------------+     +-------------+
                                               |
+-------------+     +-------------+     +-------------+
|  Result     | <-- | Interpreter | <-- |    AST      |
| Formatter   |     |  (解释执行)  |     |  (抽象语法树)|
+-------------+     +-------------+     +-------------+
       ^                                          |
       |            +-------------+               |
       +----------- | TypeChecker | <-------------+
                    | (类型检查)   |
                    +-------------+
```

### 关键特性

1. **多行输入支持**
   - 自动检测未闭合的括号/花括号
   - 缩进提示（根据块嵌套深度）
   - 续行符支持（行尾 `\`）

2. **代码补全引擎**
   - 32 个 Claw 关键字
   - 9 个内置函数
   - 动态提取用户定义的变量和函数
   - 前缀匹配补全

3. **会话管理**
   - 输入历史记录
   - 环境重置
   - 历史保存/加载

4. **调试支持**
   - 调试模式开关
   - 词法分析结果展示
   - AST 可视化
   - 执行时间测量
