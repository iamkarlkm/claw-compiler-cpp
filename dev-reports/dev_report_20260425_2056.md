## 本次开发任务

### 选择功能: RISC-V 发射器延迟绑定 (Lazy Binding)

**原因**: RISC-V 发射器是 JIT 多目标支持的关键组件，延迟绑定是实现前向跳转（forward jump）的核心技术。

### 实现内容

**文件**: `src/emitter/riscv_emitter.h` + `src/emitter/riscv_emitter.cpp`

**新增功能**:

1. **PendingJump 结构体** (~30 行)
   - 记录待回填跳转的位置、目标标签、跳转类型
   - 支持 JAL/BEQ/BNE/BLT/BGE/BLTU/BGEU/CALL 8 种跳转类型

2. **Label 类扩展** (~15 行)
   - 新增 `pending_jumps_` 向量跟踪所有引用此标签的跳转
   - 新增 `add_pending_jump()` 方法

3. **延迟绑定核心实现** (~150 行)
   - `create_pending_jump()` - 创建并记录待回填跳转
   - `resolve_pending_jumps()` - 回填所有引用指定标签的跳转
   - `compute_branch_offset()` - 计算跳转偏移量

4. **修改跳转指令支持** (~60 行)
   - 修改 `j()`, `jmp()`, `jal()`, `call()`, `jeq()`, `jne()`, `jlt()`, `jge()`, `tail()` 方法
   - 标签未绑定时自动记录待回填跳转

## 技术亮点

1. **双向追踪机制**: 标签记录所有引用它的跳转，跳转也记录目标标签，实现 O(1) 回填查找
2. **类型安全**: 使用强类型枚举区分 8 种跳转类型
3. **自动回填**: bind_label() 时自动触发所有待回填跳转的解析
