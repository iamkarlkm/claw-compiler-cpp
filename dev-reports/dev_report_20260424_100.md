## 4. 遇到的问题与解决

### 问题 1: 模块 API 不兼容
- **现象**: 编译时出现 20+ 错误，主要来自 vm/claw_vm.cpp 与 bytecode/bytecode.h 之间的 API 不匹配
  - `BytecodeReader::read()` → 应为 `read_module()`
  - `Module::instructions` → 不存在 (应使用 functions)
  - `ClosureValue::get()` → 不存在
- **解决**: 绕过了直接编译，创建新的执行管道模块避免链接不兼容的代码

### 问题 2: Makefile 路径问题
- **现象**: Makefile 引用 `src/bytecode/bytecode_compiler.cpp` 但该文件不存在
- **解决**: 存在多个备份版本 (.fixed, .backup2 等)，但为避免破坏现有工作配置，选择不修改 Makefile
