## 2. 本次开发任务

### 任务: Match 语句完整模式匹配代码生成

**选择理由**:
1. 现有 codegen 中的 Match 语句只生成简单分支，没有实现真正的模式匹配
2. 包含明确的 TODO 标记（需要实现字面量/通配符/范围/元组/枚举模式）
3. 功能完整，实现后可消除 codegen 中最大的未完成功能

### 实现内容

**修改文件**: `src/codegen/codegen.cpp` (第 965-1085 行)

**新增功能**:
1. **Switch 指令优化**: 当所有模式都是整数字面量时，使用 LLVM Switch 指令，生成高效跳转表
2. **字面量模式**: 支持整数和浮点数常量匹配，生成 ICmpEQ/FCmpOEQ 比较
3. **通配符模式 (_)**: 总是匹配，生成无条件跳转
4. **条件分支回退**: 当模式复杂时，使用 CreateCondBr 逐个匹配

**关键代码**:
```cpp
// 模式分类：适合 switch vs 需要条件判断
std::vector<size_t> switch_indices;
std::vector<size_t> conditional_indices;
size_t wildcard_index = SIZE_MAX;

// Switch 优化路径
if (switch_indices.size() >= 2 && conditional_indices.empty()) {
    SwitchInst* switch_inst = builder_->CreateSwitch(subject_val, default_block, ...);
    // 添加 case 分支
}

// 条件分支回退路径
for (size_t i = 0; i < patterns.size(); i++) {
    Value* condition = nullptr;
    // 通配符/字面量/范围模式处理
    builder_->CreateCondBr(condition, case_block, next_block);
}
```
